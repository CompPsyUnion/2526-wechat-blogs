# 2526 WeChat Blogs

2526学年 宁诺CPU公众号文章 报名 & 审核 & 提交仓库 

你可以在 [Issue 页面](https://github.com/CompPsyUnion/2526-wechat-blogs/issues) 通过新建一个 issue 提出你的公众号文章提议。

在开始前，请查阅 [https://github.com/CompPsyUnion/2526-wechat-blogs/issues/1](https://github.com/CompPsyUnion/2526-wechat-blogs/issues/1)，我们在这里提供了一个工作流程示例。请参考这个这个示例的方法提出 issue。

我们也可能通过自己创建一个 issue 然后 assign 给你来派发任务，这时你就需要新想出一个文章标题和大纲然后 Comment 到下方，我们会继续上面示例中的下一步流程。所以也请关注你的 GitHub 通知和邮箱消息。

祝工作愉快 ；）
